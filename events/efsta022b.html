【IT Professional Session 1】
「クラウドを中心とした潮流と Windows Azure概要」
講師：松崎 剛 （日本マイクロソフト株式会社）

■スライド
こちら

■受講者へのメッセージ
エフスタ Tokyo で聴講いただいた皆様、ありがとうございました。
「クラウドとは ?」というお題をいただいてかなり悩みましたが、あくまでも中立的な (1 エンジニアのスタンス) から public cloud のイメージや実体について届けさせていただき、オマケとして Windows Azure を紹介させていただきました。
今回はかなりブレーキを踏みながら説明しましたが、次回は、全力投球で「Windows Azure」を説明しますので (←私の本業)、これに懲りず、また登壇させてください。
これからも、エフスタ、応援しています！

【Fukushima All Stars Session】

「EXTREME AWS」
講師：栗原 淳一 （JAWS-UG会津兼エフスタ!!スタッフ ）

■スライド
こちら

■受講者へのメッセージ
AWSのすばらしさを十分にお伝えできなかった点、残念でなりません。
少しでも関心を持っていただけた方、もっと詳しく知りたい方は AWSユーザー会の祭典である、JAWS-DAYSにぜひお越しください。
３月15日にベルサール新宿グランドで開催予定です。 AWSの魅力とスケール、スピードを実感する一日になるはずです！http://jawsdays2014.jaws-ug.jp/
また、JAWS-UGは東京だけでなく、中央線、つくば、千葉、横浜、湘南支部と多数の支部が活発に活動しています。ぜひ一度参加いただければと思います。
2013年も残りわずかとなりましたが、まだ、困難な状況におかれている方が多数いらっしゃいます。福島へ関心を持ち続けていただき、ありがとうございます。 今後もご支援のほどよろしくお願いします。

「ビッグデータ解析手法を用いてソーシャルメディアの評価分析を実現する」
講師：石川 智史 （JAWS-UG会津兼エフスタ!!AIZUスタッフ ）

■スライド
こちら

■受講者へのメッセージ
残念ながらラブプラスの素晴らしさは解っていただけませんでしたが(笑)、今回ご紹介させていただいた、ソーシャル分析をはじめとするビッグデータ解析技術というのは、いま最も注目が集まる分野であり、今後、IT関連技術者として生きていくのであれば、避けては通れない技術となります。
クラウド、ビッグデータ、ソーシャルセキュリティ、HTML5インターフェース等、近年の凄まじい技術革新からもたらされるITの新しい潮流は、今がまさに時代の転換点をであることを物語っています。
楽しまなきゃ損です！大いに騒ぎましょう！
